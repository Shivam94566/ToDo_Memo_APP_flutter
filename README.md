# 📝 Notes & Memo Keeper

## 📌 Overview
Notes & Memo Keeper is a simple note-taking application designed to help users quickly jot down ideas and organize them with colors.  
It provides an intuitive interface for writing, categorizing, and managing notes effectively.

---

## 🎯 Objectives
- Allow users to write and save notes.
- Organize notes with color categories.
- Provide easy sorting and deletion options.
- Enable customization for better readability.

---

## ⚙️ Functional Requirements

### Notes Gallery
- Display notes in a **Staggered Grid** or list view.
- Show a snippet of note content and the date created.

### Editor Screen
- Large **TextField** for long-form writing (`maxLines: null`).
- **Color picker** (row of colored circles) to categorize notes.

### Features
- **Sorting:** Toggle between "Newest First" and "Oldest First."
- **Soft Delete:** Move notes to a "Trash" list before permanent deletion.
- **Customization:** Change font size using a **Slider**.

---

## 📥 Inputs
- **Note Title**
- **Body Text**
- **Chosen Color**

---

## 📤 Outputs
- Organized list of searchable text memos.
- Notes displayed with color categories and creation date.
- Trash list for deleted notes.
